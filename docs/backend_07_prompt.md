# Prompt 07: Configuración de CORS y Variables de Entorno

Actúa como un desarrollador Backend Semi-Senior. Ahora que la API está lista, necesito permitir que mi aplicación de Angular 21 se conecte a ella.

Por favor, proporciona:
1. Los comandos para instalar el paquete cors y sus respectivos tipos de TypeScript.
2. La configuración del middleware de CORS en el archivo principal (app.ts o server.ts) para permitir peticiones desde http://localhost:4200 (el puerto por defecto de Angular).
3. Configura los encabezados permitidos, incluyendo Content-Type y Authorization (necesario para el token JWT).
4. Asegúrate de que el puerto del servidor y la URL de la base de datos se manejen a través de un archivo .env usando la librería dotenv.

El objetivo es dejar la API disponible para ser consumida por el cliente frontend.
