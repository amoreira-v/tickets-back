# Prompt 01: Setup inicial y arquitectura de carpetas

Actúa como un Arquitecto de Software Senior. Necesito inicializar un proyecto de Backend desde cero utilizando Node.js, TypeScript y Express para una plataforma de gestión de tickets.

Por favor, proporciona:
1. Los comandos de terminal para inicializar el proyecto (npm init), instalar TypeScript y las dependencias necesarias para un entorno de desarrollo profesional (incluyendo nodemon, ts-node y eslint).
2. La configuración recomendada para el archivo tsconfig.json.
3. Una propuesta de estructura de carpetas basada en responsabilidades que incluya directorios separados para: config, controllers, services, repositories, models, routes, middlewares y utils.
4. Los comandos para instalar TypeORM y el driver de PostgreSQL para la persistencia de datos.

El objetivo es establecer una base escalable y limpia antes de comenzar con la lógica de negocio.
